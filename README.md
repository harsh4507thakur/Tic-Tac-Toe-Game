# Tic-Tac-Toe-Game
A simple Tic Tac Toe game built using HTML, CSS, and JavaScript. This project helped me understand basic DOM manipulation, event handling, and game logic in JavaScript.

Features
Two-player game (X vs O)
Displays winner when a player wins
Draw (match withdraw) condition
Restart / New Game functionality
Simple and clean UI

Technologies Used
HTML
CSS
JavaScript

How to Run the Project
Clone the repository or download the ZIP
Open the index.html file in any web browser
Start playing the game

Learning Outcome
Improved JavaScript logic building
Better understanding of DOM manipulation
Hands-on practice with CSS styling
Future Improvements
Add single-player mode (AI)
Improve UI/animations
Add score tracking
